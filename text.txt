
Hello!!! this is Node.js Server.
<br>As you may know that...

<p>
Node.js is an open source server environment
<br>Node.js is free 
<br>Node.js runs on various platforms (Windows, Linux, Unix, Mac OS X, etc.)
<br>Node.js uses JavaScript on the server
</p>
<p>
Node.js can generate dynamic page content
<br>Node.js can create, open, read, write, delete, and close files on the server
<br>Node.js can collect form data
<br>Node.js can add, delete, modify data in your database
</p>

<p>
Node.js files contain tasks that will be executed on certain events
<br>A typical event is someone trying to access a port on the server
<br>Node.js files must be initiated on the server before having any effect
<br>Node.js files have extension ".js"
</p>
